# مدونة قواعد السلوك (Code of Conduct)

## تعهدنا
نحن كأعضاء ومساهمين ومدراء لهذا المشروع، نتعهد بجعل المشاركة في مشروعنا وتطويره تجربة خالية من المضايقات للجميع، بغض النظر عن العمر أو حجم الجسم أو الإعاقة أو العرق أو الخصائص الجنسية أو الهوية الجندرية وتعبيراتها أو مستوى الخبرة أو التعليم أو الخلفية الاجتماعية والاقتصادية أو الجنسية أو المظهر الشخصي أو العرق أو الدين أو الهوية الجنسية والتوجه الجنسي.

## معاييرنا
الأمثلة على السلوكيات التي تساهم في خلق بيئة إيجابية تشمل:
- استخدام لغة مرحبة وشاملة.
- احترام وجهات النظر والتجارب المختلفة.
- تقبل النقد البناء بصدر رحب.
- التركيز على ما هو الأفضل للمجتمع.
- إظهار التعاطف تجاه أعضاء المجتمع الآخرين.

الأمثلة على السلوكيات غير المقبولة من قبل المشاركين تشمل:
- استخدام لغة أو صور جنسية والاهتمام الجنسي غير المرغوب فيه.
- التعليقات المهينة أو الشخصية أو السياسية.
- التحرش العام أو الخاص.
- نشر المعلومات الخاصة للآخرين، مثل العناوين المادية أو الإلكترونية، دون إذن صريح.
- أي سلوكيات أخرى قد تُعتبر بشكل معقول غير مناسبة في بيئة مهنية.

## مسؤولياتنا
يتحمل مديرو المشروع مسؤولية توضيح معايير السلوك المقبول ويُتوقع منهم اتخاذ إجراءات تصحيحية مناسبة وعادلة استجابة لأي حالات سلوكية غير مقبولة.

يحق لمديري المشروع إزالة أو تحرير أو رفض التعليقات والمساهمات البرمجية وتعديلات ويكي والمسائل والمساهمات الأخرى التي لا تتماشى مع مدونة قواعد السلوك هذه، أو حظر مؤقت أو دائم لأي مساهم لارتكاب سلوكيات يعتبرونها غير مناسبة أو تهديدية أو مسيئة أو ضارة.

## نطاق التطبيق
تنطبق هذه المدونة على جميع المساحات التابعة للمشروع، وتُطبق أيضاً في المساحات العامة عندما يُمثل الفرد المشروع أو مجتمعه. تشمل أمثلة تمثيل المشروع أو المجتمع استخدام بريد إلكتروني رسمي للمشروع، النشر عبر حسابات تواصل اجتماعي رسمية، أو العمل كممثل معين في حدث متصل بالإنترنت أو حدث واقعي. يجوز لمديري المشروع توضيح مفهوم وتمثيل المشروع بشكل أكبر.

## التنفيذ
يجب إبلاغ فريق المشروع عن حالات السلوك المسيء أو المزعج أو غير المقبول عن طريق التواصل عبر البريد الإلكتروني (hi@ruman.sa). سيقوم فريق المشروع بمراجعة جميع الشكاوى والتحقيق فيها والرد عليها بطريقة يراها مناسبة لحالة معينة. فريق المشروع ملزم بالحفاظ على سرية المُبلّغ عن الانتهاك.
